# BRAHMA-SCAN-LITE
Brahma Scan Lite is a self-contained advanced web intelligence platform built for OSINT researchers, security professionals and bug bounty hunters. It works on Android as well as iPhone along with all kind of devices and Important Operating Systems like Windows, Linus & MAC. It combines 12 search engines (Google, Bing, Shodan, GitHub, Yandex, and more), a 100+ entry Google Dork library across 14 categories and a visual dork builder with 12 operators. Six AI providers — Claude, Gemini, ChatGPT, DeepSeek, Qwen and Mistral — power deep research, dork generation and target analysis. Features include Multi-Engine simultaneous scanning, a full operators reference, session history, export tools and a self-diagnostic watchdog. Everything runs from a single downloadable HTML file — no installation required. 6 AI Providers - 🤖 Claude (Anthropic), ✨ Gemini (Google), 🟢 ChatGPT (OpenAI), 🔵 DeepSeek, 🟠 Qwen (Alibaba) and 🌊 Mistral (EU). Dork Builder — 12-field visual constructor (site, filetype, inurl, intitle, intext, exact phrase, date range, language, TLD, exclude, allinurl, allintitle) AI Generator — Single dork, Multi-dork set (6 at once), and Target Analysis mode powered by AI Deep Research — Full OSINT intelligence brief with animated progress bar, exportable as HTML report Search Engines (12): Google, Bing, DuckDuckGo, Yandex, Startpage, Yahoo, Twitter/X, GitHub, Google Books, Google Images, Google Video, Shodan Multi-Scan — Launch 12 engines simultaneously with one click GHDB Library — 100+ dorks across 14 categories with live search filter Results — Running log of all launched scans Provider selector — click any AI logo to switch, API key saves to session storage Model picker — choose specific model per provider Live key status — green dot when key is valid, show/hide toggle 4 Research modes — Full OSINT Brief, Dork Strategy Only, Vulnerability Focus, Multi-AI Compare ⚖️ Compare AIs — runs your topic through ALL configured providers simultaneously, side-by-side results Save Dorks button — auto-extracts dork queries from research output into your saved collection All AI Generator tab functions (dork gen, multi-dork, analyze) also use the selected provider
